

## Start Project

Run `npm start` to start project.

## Build 
Run `npm run build` to build project.
 

## database setup
edit database setup config in repository/database.js file
eg:
const pool = mysql.createConnection({
    host: 'localhost',
    database: 'demodb',
    user: 'root',
    password: 'root'
});



## swagger Docs
Run `swagger project edit` to edit api docs.
